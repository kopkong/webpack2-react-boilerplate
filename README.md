# webpack2-react-boilerplate
Webpack2-react-boilerplate. Using express and webpack-dev-middleware for development.

Webpack2-react的模板，没有使用react-hot-loader 和 webpack-dev-server 是为了开发方便，2是从原来webpack1迁移过来比较方便。
